# HcsCloseComputeSystem

## Description

Closes a handle to a compute system

## Syntax

```cpp
void WINAPI
HcsCloseComputeSystem(
    _In_ HCS_SYSTEM computeSystem
    );

```

## Parameters

|Parameter     |Description|
|---|---|---|---|---|---|---|---|
|`computeSystem`| Receives a handle to the compute system.|
|    |    |

## Return Values

None
