# HcsDestroyLayer

## Syntax

### Parameters
|Parameter     |Description|
|---|---|---|---|---|---|---|---| 
|`layerPath`| Path of the layer to delete|
|    |    | 



## Return Values

The function returns `S_OK` on success. `HRESULT` error code for failures to delete the layer.

## Remarks
This function deletes a layer from the host.
