# HcsCloseProcess

## Syntax
```C
void WINAPI
HcsCloseProcess(
    _In_ HCS_PROCESS process
    );

```

### Parameters
|Parameter     |Description|
|---|---|---|---|---|---|---|---| 
|`process`| Process handle to close|
|    |    | 



## Remarks
This function closes the handle to a process in a compute system.
