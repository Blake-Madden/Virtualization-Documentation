# HDV_DEVICE_TYPE

## Description

Discriminator for the Emulated device type.

## Syntax

```C++
typedef enum HDV_DEVICE_TYPE
{
    HdvDeviceTypeUndefined = 0,
    HdvDeviceTypePCI
} HDV_DEVICE_TYPE;
```
